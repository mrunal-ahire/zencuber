# Zen Cuber
> A sleek webapp for speedcubers.

![](http://i.imgur.com/sxXkTCJ.png)

## Developing

### Install the dependencies:
```
$ npm install
```

### Start the app
```
$ npm start
```
### Lint and Typecheck
```
npm test
```
